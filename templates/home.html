Weather Data API

To get the average temperature of a day in a station

URL FORMAT : http://127.0.0.1:5000/station/date=date

URL EXAMPLE : http://127.0.0.1:5000/10/date=2024-5-11

EXAMPLE OUTPUT : {'station_id':10, 'station': 'STOCKHOLM', 'date': '2024-5-11', 'temperature': 2.3, 'year_mean': 7.98, 'month_mean': 3.56}

(year_mean and month_mean gives the mean temperature value of the selected station in the respective year and month of the given date)

To get the whole temperature data of a year in a station

URL FORMAT : http://127.0.0.1:5000/station/year=date

URL EXAMPLE : http://127.0.0.1:5000/10/year=2000

EXAMPLE OUTPUT : [ { "DATE": "20000101", "STAID": 10, "TG": -2.3 }, { "DATE": "20000102", "STAID": 10, "TG": 1.3 },............................. ]

(DATE format id - 'YYYYMMDD')

To get the average temperature of each day for a station

URL FORMAT : http://127.0.0.1:5000/station/

URL EXAMPLE : http://127.0.0.1:5000/10/

EXAMPLE OUTPUT : [ { "DATE": 17560101, "STAID": 10, "TG": -8.7 }, { "DATE": 17560102, "STAID": 10, "TG": -9.2 },........................]

(DATE format is - 'YYYYMMDD')
